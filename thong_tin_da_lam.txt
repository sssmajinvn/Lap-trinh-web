4. CÔNG NGHỆ SỬ DỤNG
----------------------
 - ASP.NET Core 8.0 MVC (Razor Views - .cshtml)
 - Vanilla CSS (biến CSS Custom Properties)
 - Vanilla JavaScript (fetch API, AJAX, localStorage)
 - Font: Outfit (Google Fonts)
 - Icons: FontAwesome 6.4.0
 - Backend: Các API có sẵn (/api/movies, /api/bookings, /api/reviews/...)
 - Auth: JWT Bearer Token (lưu vào localStorage)

5. CÁCH CHẠY VÀ KIỂM TRA
--------------------------
 Bước 1: Mở terminal tại thư mục TicketBookingApi
 Bước 2: Chạy lệnh: dotnet run
 Bước 3: Mở trình duyệt và truy cập:
   - Trang chủ:      http://localhost:5175/
   - Tìm kiếm phim:  http://localhost:5175/Home/Search
   - Chi tiết phim:  http://localhost:5175/Home/Details/{maphim}
   - Đặt vé:         http://localhost:5175/Home/Booking?movieId={maphim}
   - Swagger API:    http://localhost:5175/swagger

 Lưu ý port:
   - dotnet run (profile http)  -> http://localhost:5175/
   - dotnet run --launch-profile https -> https://localhost:7100/
   - Visual Studio -> chọn profile http hoặc https trên thanh công cụ
