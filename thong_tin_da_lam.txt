1. MỤC TIÊU
-----------
Tạo phần giao diện người dùng (UI) bằng ASP.NET Core MVC (Razor Views)
theo chuẩn thiết kế Dark Cinema Mode (form-ui.md), đồng thời tích hợp 
các API hệ thống được cập nhật mới nhất từ GitHub mà không ảnh hưởng 
đến code backend cũ.

2. CÁC FILE ĐÃ TẠO MỚI & CẬP NHẬT TRONG DỰ ÁN
---------------------------------------------

[A] CẤU HÌNH & HỆ THỐNG
 - TicketBookingApi/Program.cs
   + Đã cấu hình AddControllersWithViews() hỗ trợ đồng thời cả API và MVC Views.
   + Đã tích hợp middleware static files app.UseStaticFiles().
   + Đã thêm Route default map tới HomeController.
   + Đã thêm middleware redirect /index.html -> / để fix lỗi tự động redirect của IIS Express.
   + Cấu hình RoutePrefix = "swagger" để nhường đường dẫn mặc định "/" cho trang chủ MVC.

 - TicketBookingApi/Properties/launchSettings.json
   + Đã sửa launchUrl từ "swagger" thành "" cho tất cả các profile (http, https, IIS Express) để mở trực tiếp trang chủ MVC khi khởi chạy.

 - TicketBookingApi/wwwroot/PaymentWaiting.html [CẬP NHẬT]
   + Đã chép file PaymentWaiting.html từ thư mục gốc repository vào thư mục wwwroot để server có thể phục vụ tĩnh (tránh lỗi 404 khi người dùng chuyển tiếp từ trang chọn ghế sang cổng thanh toán).

[B] CONTROLLER
 - TicketBookingApi/Controllers/HomeController.cs [CẬP NHẬT]
   + Index()             -> Trang chủ
   + Details(id)        -> Chi tiết phim
   + Search()           -> Tìm kiếm phim
   + Booking()          -> Chọn suất chiếu
   + Seats()            -> Chọn ghế ngồi & combo bắp nước (mới từ GitHub)
   + Success()          -> Đặt vé thành công (mới từ GitHub)
   + GetSearchMovies()  -> API tìm kiếm phim nâng cao
   + GetAllGenres()     -> API lấy toàn bộ thể loại phim

 - TicketBookingApi/Controllers/AdminViewController.cs [MỚI]
   + GET  /Admin hoặc /Admin/Dashboard     -> View Tổng quan quản trị
   + GET  /Admin/Login                     -> View Đăng nhập quản trị
   + GET  /Admin/Movies                    -> View Quản lý danh sách phim
   + GET  /Admin/Showtimes                 -> View Quản lý lịch chiếu
   + POST /Admin/api/movies/update/{id}    -> Cập nhật thông tin phim (nội bộ MVC)
   + POST /Admin/api/movies/delete/{id}    -> Xóa phim hoặc ẩn phim (nội bộ MVC)
   + POST /Admin/api/showtimes/update/{id} -> Cập nhật lịch chiếu (nội bộ MVC)
   + POST /Admin/api/showtimes/delete/{id} -> Xóa lịch chiếu (nội bộ MVC)

 - Dashboard Admin & Thống kê: Các controller mới trong thư mục Controllers/Admin (Auth, Concession, Movie, Showtime, Stats, User, Voucher) phục vụ công tác quản lý của admin.
 - Fix lỗi cơ sở dữ liệu: Cập nhật định dạng ngày giờ PostgreSQL và logic kiểm tra phim tồn tại.
 - Sửa lỗi & Tối ưu hóa API Admin (Mới):
   + Tạo mới `TheaterAdminController.cs` cung cấp `/api/admin/theaters/rooms` (lấy danh sách phòng và rạp tương ứng).
   + Cập nhật `ShowtimeAdminController.cs` (GetAll) trả thêm tên phim, tên phòng, tên rạp từ các Navigation Properties.
   + Sửa lỗi so sánh kiểu dữ liệu date = text trong `StatsAdminController.cs` (DashboardSummary) bằng cách ép kiểu CAST(parameter AS date).
   + Cập nhật View `Dashboard.cshtml` liên kết chính xác các thuộc tính DTO mới (totalRevenueToday, ticketSoldToday, totalCustomersToday).

4. CÁCH KHỞI CHẠY DỰ ÁN
-----------------------
 Bước 1: Mở Terminal tại thư mục TicketBookingApi
 Bước 2: Chạy dự án bằng lệnh:
         dotnet run
         (Hoặc chạy qua Visual Studio bằng cách nhấn F5)
 Bước 3: Mở trình duyệt và truy cập các cổng:
   - Trang chủ MVC:       http://localhost:5175/
   - Trang Swagger API:   http://localhost:5175/swagger
   - Cổng bảo mật HTTPS: https://localhost:7100/
   - Admin Panel:         http://localhost:5175/Admin  (yêu cầu đăng nhập)
